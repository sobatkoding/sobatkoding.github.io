Tentang Sobat Koding

Blog ini sesuai dengan tagline-nya berusaha ingin menjadi teman dalam kegiatan koding pada developer, karena belajar bareng kadang lebih seru, lebih mudah dan menyenangkan.

Dengan kemampuan koding yang tidak banyak ‘BELUM JAGO KODING’ mudah-mudahnya konten blog ini tidak salah dan menyesatkan, tapi jadi ajang untuk meningkatkan kemampuan, dan harapan saya akan ada banyak masukan, saran dan perbaikan dari para pembaca.

Siapa saya? Nama saya Taufik, pengajar di salah satu sekolah swasta di Jakarta, belajar web programming secara mandiri, yang saat ini masih berkutat di seputaran, HTML, CSS, JS, dan PHP. Web framework yang saya sering gunakan dalam project-project saya adalah: Laravel, Kohana, Angular JS, Vue JS, Node JS, Express JS, dan juga Bootstrap.
